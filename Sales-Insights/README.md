# 📊 Sales Insights Dashboard

## 🧠 Project Overview
This project focuses on building an interactive Sales Insights Dashboard for Atliq Hardware, a computer hardware manufacturer operating in a rapidly growing market.

The goal was to transform raw sales data into meaningful insights and enable data-driven decision-making for business growth.

---

## 🎯 Problem Statement
Atliq Hardware faced difficulty in tracking sales performance due to:
- Reliance on verbal reports from regional managers (often biased)
- Large and complex Excel files that were hard to analyze
- Lack of a centralized system for clear business insights

---

## 🎯 Objective
To develop an interactive dashboard that:
- Tracks revenue and sales performance
- Identifies key markets, customers, and products
- Provides actionable insights for better decision-making

---

## 🛠️ Tools & Technologies
- SQL (MySQL) → Data exploration and analysis  
- Power BI → ETL, Data Modeling, Visualization  
- Power Query → Data cleaning and transformation  
- DAX → KPI calculations  

---

## 🔄 Project Workflow

1. **Data Analysis (SQL)**
   - Performed data exploration using SQL
   - Used JOIN, GROUP BY, and aggregations
   - Identified data inconsistencies and patterns

2. **ETL Process (Power BI)**
   - Imported data into Power BI
   - Cleaned and transformed using Power Query
   - Handled missing and inconsistent values

3. **Data Modeling**
   - Created relationships between tables
   - Built a **Star Schema** (Fact + Dimension tables)
   - Improved performance and accuracy

4. **Dashboard Development**
   - Designed interactive visuals
   - Added filters for dynamic analysis
   - Created KPI measures using DAX

---

## 📊 Dashboard Features

- Revenue Overview (985M)
- Sales Quantity (2M)
- Revenue by Market
- Sales Quantity by Market
- Revenue Trend Analysis (2017–2020)
- Top 5 Customers
- Top 5 Products
- Interactive Filters (Year, Month)

---

## 🔍 Key Insights

- Delhi NCR generates the highest revenue (~520M)
- A small group of customers contributes a major portion of total revenue
- Some markets are underperforming and need improvement
- Revenue trends show fluctuations over time

---

## 📈 Business Impact

- Enabled data-driven decision-making  
- Improved visibility of business performance  
- Identified growth opportunities  
- Potential to increase revenue by ~7%  

---

## 🚀 Conclusion

This project demonstrates an end-to-end data analytics workflow:
- SQL for data exploration  
- Power BI for ETL, data modeling, and visualization  

It highlights how raw data can be transformed into actionable business insights.
